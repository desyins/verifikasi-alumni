Aplikasi Verifikasi Alumni (Graduate Verification Application)

A website-based application built with Laravel that allows employers, HR proffesionals and external parties to verify graduate certificates online.

Features
Certificate verification form.
Certificate information page showing verification results
Integration with internal REST APIs
API-driven architecture without a separate database

Technologies
PHP
Laravel
REST API
Bootstrap

Installation
Clone the repository.
Install dependencies with composer install.
Configure environment variables.
Run the application using php artisan serve.
